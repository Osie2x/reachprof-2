About

Lee Nakamura joined Lakeshore College in 2018 after eight years in software development and technical consulting. Lee's background is primarily applied: large-scale data pipeline engineering, backend web development, and developer tooling. At Lakeshore, Lee focuses on preparing students for practical careers in software development and data analysis.

Teaching Areas

Lee teaches across the core computing curriculum with emphasis on practical programming skills, data management, and software engineering fundamentals. Areas of focus include:

  • Data structures and algorithms
  • Database systems and SQL
  • Web application development
  • Software engineering practices
  • Introduction to data analytics with Python

Student Projects

Lee is open to supervising applied capstone projects in the final year of the CIS diploma program. Projects must have a practical software development or data analysis component. Students interested in capstone supervision should email Lee with a one-paragraph project description in their second-to-last term.

Topics Lee has supervised in the past include: inventory management systems, web-based dashboards for local nonprofits, data cleaning and visualization pipelines, and REST API development. Research-oriented projects are generally outside the scope of supervision offered here.
